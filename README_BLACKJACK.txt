===== ЭТАП 1: ВЫПОЛНИТЬ SQL В SUPABASE =====

ВАЖНО! Это нужно сделать ИМ один раз!

Шаг 1️⃣: Откройте Supabase
  - Перейдите на https://supabase.com
  - Откройте ваш проект

Шаг 2️⃣: Откройте SQL Editor
  - В левой панели нажмите "SQL Editor"
  - Нажмите кнопку "New query" (создать новый запрос)

Шаг 3️⃣: Скопируйте SQL
  - Откройте файл "SETUP_FRESH.sql" в редакторе кода
  - Выделите ВСЕ содержимое (Ctrl+A)
  - Скопируйте (Ctrl+C)

Шаг 4️⃣: Вставьте в Supabase
  - Вернитесь в Supabase SQL Editor
  - Вставьте SQL (Ctrl+V) в поле запроса

Шаг 5️⃣: Выполните
  - Нажмите синюю кнопку "RUN"
  - Подождите ✅ "Success"

Шаг 6️⃣: Проверьте
  - В левой панели нажмите "Table Editor"
  - Должны быть таблицы:
    ✅ profiles
    ✅ blackjack_rooms
    ✅ blackjack_players
    ✅ blackjack_history

===== ЭТАП 2: ИСПОЛЬЗУЙТЕ BLACKJACK =====

Откройте браузер, перезагрузитесь (Ctrl+R или F5)

Зайдите в игру → вкладка "Multiplayer Blackjack"

✅ ВЫ ДОЛЖНЫ ВИДЕТЬ:
  - Ваш баланс в верхнем углу (например: Balance: 5000)
  - Кнопка "Create Room"
  - Список доступных комнат

===== КАК ИГРАТЬ =====

🎬 ЛИДЕР (тот кто создаёт комнату):
  1. Вводит минимальную ставку (например 10)
  2. Нажимает "Create Room"
  3. Видит ID комнаты (первые 8 символов)
  4. Говорит это ID друзьям

👥 ДРУГИЕ ИГРОКИ (присоединяются):
  1. Получают ID комнаты от лидера
  2. Вводят этот ID в поле "Join Existing Room"
  3. Нажимают "Join Room"
  4. Видят себя в списке в "Waiting Room"

🎮 НАЧАЛО ИГРЫ:
  1. Когда все присоединились → Лидер видит кнопку "Start Game"
  2. Лидер нажимает "Start Game"
  3. Все игроки видят экран "Place Your Bet"
  4. Каждый вводит свою ставку (≥ минимальной)
  5. Когда все поставили → раздаются карты
  6. По очереди: Hit (карта), Stand (хватит), Double Down (удвоить)
  7. Дилер ходит автоматически
  8. Результаты и выплаты!

===== ЕСЛИ ОШИБКА =====

❌ "Could not get profile - Перезагрузитесь"
  → Значит таблица profiles не создана
  → Выполните SQL (шаги выше)
  → Перезагрузите браузер (Ctrl+R)

❌ "Cannot create room"
  → Проверьте что вы залогинены
  → Закройте консоль браузера (F12) чтобы смотреть ошибки

❌ "RPC function does not exist"
  → SQL не полностью выполнен
  → Попробуйте выполнить SQL ещё раз

===== СОВЕТЫ =====

💡 Откройте консоль браузера (F12 → Console) чтобы видеть логи:
  - Зелёные логи ✅ = всё хорошо
  - Красные логи ❌ = есть проблема

💡 Если всё зависает - перезагрузите браузер (Ctrl+R)

💡 Проверьте что у всех есть интернет соединение

Готово! 🚀 Успехов в игре!
